{# pdf-dispatch — single-page application ========================================= Rendered by the index() route (app.py) via Flask/Jinja2. Only two values are injected server-side: - lang : active language code ("fr" or "en"), used for and window.LANG. - i18n_json : complete translation dictionary for the active language (splitter/i18n/.json), injected into window.I18N and consumed by t('key', {...}) in static/js/app.js. CSS (static/css/style.css) and JS (static/js/app.js) are static files with no templating — all dynamic logic goes through /api/... calls from app.js. #} pdf-dispatch
